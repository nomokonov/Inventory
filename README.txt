Приложение для своих нужд.
Для проведения инвентаризации.
Позволяет загружать список имущества из *.xlsx файла 
делать перемещения имущества по  подраздедениям через форму поиска.
По результатам инвентаризации формируются документы на премещения (Унифицированная форма № ТОРГ-13)  в формате *.xlsx
SpringBoot авторизация - Win ActiveDirectory. Hibernate JPA, PostgreSQL, bootstrap html, Apache POI,
